## fish-man
操作系统小学期项目

## 环境要求
+ **ubuntu 32** 位系统
+ **bochs** 环境

## 使用说明
+ 确保 **git环境**
+ 在工作空间克隆项目

```
$ git init
$ git clone https://github.com/TalkWIthKeyboard/fish-man.git
```

+ 确保 **gcc环境**
+ 进入fish-man解压img文件

```
$ cd fish-man
$ tar -xvf 80m.img.tar.gz
```
+ 编译项目


```
$ make image
```
+ 启动bochs进入虚拟机

```
$ bochs
```
+ 输入c启动虚拟机
+ 启动后通过**F2**切换shell窗口
+ 输入**help**查看shell指令即可使用
